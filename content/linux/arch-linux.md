- when trying to moung from boot disk
  
  - to decrypt: cryptsetup open --type luks /dev/nvme0n1p2 cryptroot
  
  - mount -t auto /dev/mapper/cryptroot  /mnt

- mount boot file system
  
  - mount -t auto /dev/nvme0n1p1 /mnt/boot
